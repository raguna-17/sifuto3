# 📄 求人アプリ

## 🧭 概要

本アプリは、求人への応募情報をシンプルに記録・確認できるWebアプリです。

企業名と職種を入力し応募ボタンを押すことで、応募履歴として一覧に表示されます。

基本的なCRUD操作を通じて、応募情報を管理できるシンプルな構成になっています。

---

## 🎯 解決する課題

求人応募の記録を手軽に残したい場合、

- メモやスプレッドシートに手動で記録するのが面倒
- 応募履歴をすぐに確認できない

といった不便さがありました。

本アプリでは、最低限の入力で応募情報を記録し、一覧で簡単に確認できるようにしています。

## 🧱 アーキテクチャ設計

保守性と拡張性を重視し、レイヤードアーキテクチャを採用しています。

router → service → model → schema

ドメインごとに責務を分離し、機能追加や変更に強い構造としています。

## 🔐 認証設計

JWTベースのステートレス認証を採用しています。

ログイン時にJWTを発行
トークンからユーザーを識別
DBと照合して認証を検証

パスワードはArgon2でハッシュ化し、安全性を確保しています。

Dockerによる環境統一と再現性の確保
