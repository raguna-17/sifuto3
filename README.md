# 📄 Job Application Management API
## 🧭 概要

本アプリケーションは、就職・転職活動における応募管理を一元化するためのWebアプリです。

実際の転職活動の中で、応募先企業や選考状況がスプレッドシートや複数サービスに分散し、進捗管理が煩雑になるという課題を強く感じたことをきっかけに開発しました。

「どの企業に、どのステータスで、次に何をすべきか」を一目で把握できる状態を目指し、応募情報を中心としたデータ構造で設計しています。


https://kyuujin-bakku.onrender.com

## 🎯 解決する課題

就職・転職活動において、以下のような問題が発生しやすいと感じました：

応募情報が複数のサービス・メモに分散し管理が煩雑になる
選考状況（応募・面接・内定など）の全体像が把握しづらい
次に取るべきアクションが不明確になりやすい

本アプリでは、これらを「応募情報の一元管理」によって解決します。

## 🔐 認証設計

JWTベースのステートレス認証を採用しています。

ログイン時にJWTを発行
トークンからユーザーを識別
DBと照合して認証を検証

パスワードはArgon2でハッシュ化し、安全性を確保しています。

## 🧭 このプロジェクトについて

このアプリは単なるCRUDアプリではなく、
「転職活動の情報管理をどれだけストレスなくできるか」 を軸に設計しています。

実体験ベースの課題を起点にしているため、
プロダクト設計とバックエンド設計の両方を意識した構成になっています。
